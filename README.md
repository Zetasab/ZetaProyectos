# ZetaProyectos

Coleccion de proyectos pequenos y experimentos front-end hechos con **HTML**, **CSS** y **JavaScript**.

Este repositorio funciona como un laboratorio visual: aqui vas a encontrar ideas rapidas, componentes reutilizables, mini juegos y efectos para practicar, inspirarte o integrar en tus propios proyectos.

## Que hay aqui

- **Snippets y animaciones CSS**: loaders, menus, efectos de texto y microinteracciones.
- **Interacciones con JavaScript**: scroll effects, navbar dinamica, contadores y comportamiento responsive.
- **Mini proyectos y juegos**: ejemplos listos para abrir y probar.
- **Web demos**: secciones de paginas y experimentos de UI.

## Estructura principal

```text
ZetaProyectos/
	repositories/
		css/        -> animaciones, navs, loaders, efectos visuales
		js/         -> interacciones de scroll, formularios, componentes dinamicos
		games/      -> mini juegos y pruebas ludicas
		websites/   -> demos web mas completas
```

## Como usarlo

1. Entra a la carpeta del ejemplo que quieras revisar.
2. Abre su `index.html` en el navegador.
3. Explora y modifica archivos HTML/CSS/JS para adaptarlos a tu estilo.

Tambien puedes abrir el repositorio completo en VS Code y lanzar cada demo localmente.

## Objetivo del repo

Mantener una biblioteca de proyectos pequenos, claros y visuales que sirvan para:

- practicar front-end,
- prototipar ideas rapido,
- y reutilizar componentes en otros proyectos.

---

Si te sirve, puedes usar cualquier ejemplo como base y personalizarlo a tu gusto.
